Wilkommen zur Minigames-Webseite!



Browser Spiele, die oft auch als Minigames bezeichnet werden, sind kurze und einfache Spiele, die innerhalb eines Browsers (z.B. Edge, Chrome, Firefox) gespielt werden. Oftmals werden sie in Javascript und HTML geschrieben oder verwenden ein Framework, welches das Programmieren des Spiels erleichtert.

Auch relativ komplexe Spiele lassen sich mit solchen Tools erstellen, sofern man die Zeit und die Vision dafür hat. Wenn man bedenkt, dass das erste Videospiel, Pong, im Jahre 1958 erstellt wurde - als 10 Jahre zuvor ein einfacher Taschenrecher eine ganze Halle füllen würde (Abbildung links), ist das Erreichen einer Norm für die heutige Rechenleistung schon bei einfachen Geräten eine beeindruckende Errungenschaft.

Hier werden einige selbstgemachte Browser Spiele angeboten. Bisher gibt es nur ein funktionstüchtiges Minigame - Platformer. Das erstellen eines zweiten Spiels wurde schon begonnen, allerdings ist dieses noch nicht abgeschlossen. Aber auch dieses Spiel kann zumindest angetestet werden.


Wähle "Spielübersicht" an, um eine Liste der verfügbaren Spiele zu erhalten, oder verwende eine der Schaltflächen, um direkt einzusteigen.
